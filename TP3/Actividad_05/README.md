# Búsqueda informada: Voraz y A* en el mapa de Rumania

Este script implementa los algoritmos de búsqueda Voraz y A* para encontrar el camino óptimo desde Arad a Bucarest en el mapa de Rumania, utilizando la heurística de distancia en línea recta.

- `busqueda_voraz`: Algoritmo de búsqueda voraz con heurística.
- `busqueda_a_estrella`: Algoritmo A* con heurística.
- Se compara el tiempo de ejecución de ambos algoritmos.

Para ejecutar:
```bash
python solucion.py
```

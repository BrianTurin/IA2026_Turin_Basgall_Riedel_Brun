# IA2026 — Inteligencia Artificial
**Integrantes:** Turin · Basgall · Riedel · Brun

Repositorio de actividades de la materia **Inteligencia Artificial** de la carrera Ingeniería en Sistemas de Información.

---

## Estructura del repositorio

```
Actividades/
├── Actividad_01/
│   ├── README.md      # Enunciado y descripción de la actividad
│   └── solucion.py    # Implementación en Python (o pseudocódigo)
├── Actividad_02/
│   └── ...
└── ...
```

Cada carpeta dentro de `Actividades/` corresponde a una actividad de la materia y contiene:

| Archivo | Descripción |
|---------|-------------|
| `README.md` | Enunciado, consignas y explicación de la actividad |
| `solucion.py` | Código Python con la solución (cuando aplica) |

> **Nota:** Las actividades que usan pseudocódigo se documentan directamente en el `README.md` de la actividad correspondiente.

---

## Cómo ejecutar una actividad

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/BrianTurin/IA2026_Turin_Basgall_Riedel_Brun.git
   cd IA2026_Turin_Basgall_Riedel_Brun
   ```

2. Navegar a la actividad deseada:
   ```bash
   cd Actividades/Actividad_01
   ```

3. Leer el enunciado:
   ```bash
   cat README.md
   ```

4. Ejecutar el código (si aplica):
   ```bash
   python solucion.py
   ```

---

## Convenciones

- Los nombres de las carpetas siguen el formato `Actividad_NN` (número de dos dígitos).
- Cada actividad documenta su propio enunciado y la solución en su carpeta.
- Se utiliza Python 3 para el código ejecutable.
